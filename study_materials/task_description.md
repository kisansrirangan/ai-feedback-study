Write a Python function called `count_vowels` that takes a string as input and returns the number of vowels in the string.

Count both lowercase and uppercase vowels.

Examples:
- `count_vowels("hello")` → `2`
- `count_vowels("APPLE")` → `2`
- `count_vowels("xyz")` → `0`
- `count_vowels("Education")` → `5`
